Setting Up Git

Installation

Windows

  1. install Chocolatey
  2. choco install git
choco install cmder -pre
choco install sublimetext3
choco install sublimetext3.powershellalias
  3. install Visual C++ Redistributable

Mac

  1. install Sublime Text 3
  2. install Brew
  3. ln -s "/Applications/Sublime Text.app/Contents/SharedSupport/bin/subl" /usr/local/bin/subl
brew install git
subl ~/.bashrc
  4. Enter export PATH=/user/local/bin:$PATH into the file and save
  5. git config --global user.name "YOUR NAME"
git config --global user.email "YOUR EMAIL"
git config --global core.editor "subl -n -w"
git config --global color.ui true
git config --global push.default simple

Setting up Github

Windows

  1. choco install cygwin
choco install cyg-get
cyg-get openssh
  2. Follow Github's instructions for setting up your SSH keys, but use C:\tools\cygwin\bin\ssh-keygen instead of ssh-keygen

Mac

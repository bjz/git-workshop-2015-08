Setting Up Git

Before getting started with Git, we first need to a bit of setup. We will be installing a bunch of things as well as Git, like a package manager and an editor, the value of which may be unclear at first, but rest assured that these will serve you well as you start to use the command line more and more in your daily workflow.

Installation

Windows

  1. Install Chocolatey
  2. In an administrator-enabled command prompt, run the following commands: 
    choco install git                            # Git!
choco install sublimetext3                   # The editor we will be using
choco install cmder                          # A prettier terminal than cmd.exe :)
    Note: If you see an error when installing cmder, you might need to install Visual C++ Redistributable.
  3. Add the subl command to your PATH. (See this blog post for help)
  4. Check if everything installed correctly by opening up the Cmder application that you installed before, then running: 
    git help   # This should print out a help message for Git
subl .     # Sublime text should open in the current working directory
  5. Suggested Sublime Text settings (under Preferences > Settings - User): 
    {
  "default_line_ending": "unix",
  "highlight_line": true,
  "ignored_packages": ["Vintage"],
  "preview_on_click": false,
  "scroll_past_end": true,
  "translate_tabs_to_spaces": true,
  "trim_trailing_white_space_on_save": true
}
  6. We'll now add some handy configuration stuff to our ~/.gitconfig: 
    git config --global user.name "YOUR NAME"
git config --global user.email "YOUR EMAIL"
git config --global core.autocrlf true     # For more info, see Dealing with line endings
git config --global core.editor subl -n -w
git config --global color.ui true
git config --global push.default simple

Mac

  1. Install Sublime Text 3
  2. Suggested Sublime Text settings (under Sublime Text > Preferences > Settings - User): 
    {
  "highlight_line": true,
  "ignored_packages": ["Vintage"],
  "preview_on_click": false,
  "scroll_past_end": true,
  "translate_tabs_to_spaces": true,
  "trim_trailing_white_space_on_save": true
}
  3. Install Brew
  4. ln -s "/Applications/Sublime Text.app/Contents/SharedSupport/bin/subl" /usr/local/bin/subl
brew install git
subl ~/.bashrc
  5. Enter export PATH=/user/local/bin:$PATH into the file and save
  6. We'll now add some handy configuration stuff to our ~/.gitconfig: 
    git config --global user.name "YOUR NAME"
git config --global user.email "YOUR EMAIL"
git config --global core.autocrlf input      # For more info, see Dealing with line endings
git config --global core.editor subl -n -w
git config --global color.ui true
git config --global push.default simple

Setting up Github

Windows

  1. choco install cygwin
choco install cyg-get
cyg-get openssh
  2. Follow Github's instructions for setting up your SSH keys, but use C:\tools\cygwin\bin\ssh-keygen instead of ssh-keygen.

Mac

  1. Follow Github's instructions for setting up your SSH keys.

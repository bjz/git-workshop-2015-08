Setting Up Git

Before getting started with Git, we first need to a bit of setup. We will be installing a bunch of things as well as Git, like a package manager and an editor, the value of which may be unclear at first, but rest assured that these will serve you well as you start to use the command line more and more in your daily workflow.

Installation

Windows

  1. Install Chocolatey
  2. In an administrator-enabled command prompt, run the following commands: 
    choco install git                            # Git!
choco install cmder -pre                     # A prettier terminal than cmd.exe :)
choco install sublimetext3                   # The editor we will be using
choco install sublimetext3.powershellalias   # A handy `subl` alias for opening sublime text
  3. Install Visual C++ Redistributable (this should allow you to run cmder)
  4. Check if everything installed correctly by opening up the Cmder application that you installed before, then running: 
    git help   # This should print out a help message for Git
subl .     # Sublime text should open in the current working directory

Mac

  1. Install Sublime Text 3
  2. Install Brew
  3. ln -s "/Applications/Sublime Text.app/Contents/SharedSupport/bin/subl" /usr/local/bin/subl
brew install git
subl ~/.bashrc
  4. Enter export PATH=/user/local/bin:$PATH into the file and save
  5. We'll now add some handy configuration stuff to our ~/.gitconfig: 
    git config --global user.name "YOUR NAME"
git config --global user.email "YOUR EMAIL"
git config --global core.editor "subl -n -w"
git config --global color.ui true
git config --global push.default simple

Setting up Github

Windows

  1. choco install cygwin
choco install cyg-get
cyg-get openssh
  2. Follow Github's instructions for setting up your SSH keys, but use C:\tools\cygwin\bin\ssh-keygen instead of ssh-keygen.

Mac

  1. Follow Github's instructions for setting up your SSH keys.

## Intro to version control

Most common workflow:

1. create
2. edit
3. save
4. goto 2

```
            +------+          +--------------+
O- create ->| File |-- edit ->| Altered file |
            +------+          +--------------+
               ^                      |
               |                      |
               +-------- save --------+

```
